# 空域管理系统更新记录

## 2026-03-20 更新

### 新增功能：CNS设备图层

在"全要素保障层"下新增三个子类别：

#### 1. 通讯设备
- 包含8个5G基站设备
- 颜色标识：绿色 (emerald-500)
- 图标：📡 (lucide:radio)

#### 2. 导航设备
- 包含5个CORS连续运行参考站
- 颜色标识：蓝色 (blue-500)
- 图标：🧭 (lucide:compass)

#### 3. 监视设备
- 包含4个监视雷达设备
- 颜色标识：紫色 (purple-500)
- 图标：📡 (lucide:radar)

### 交互功能

1. **树节点点击** → 地图飞到对应设备位置
2. **地图设备点击** → 显示设备详情信息
3. **双向联动** → 树节点和地图状态同步

### 数据来源

- 通讯设备：data/通讯设备.geojson
- 导航设备：data/导航设备.geojson
- 监视设备：data/监视雷达.geojson

### 技术实现

所有设备数据已内嵌到 `cesium_airspace_standalone.html` 中，使用Cesium的Point和Label进行可视化，支持file://协议直接访问。
