### 字段说明

| 字段 | 类型 | 说明 |
|-----|------|-----|
| lon | float | 经度（度） |

| grid_code | string | 北斗格网编码 |
| per_src | object | 各信号源贡献值（仅综合模式有） |

## 使用方法

### 在网页中使用

1. **默认加载**：网页启动时自动加载 `data/templates/001/` 数据

2. **切换数据源**：
   - 点击左上角"数据源选择"下拉框
   - 选择不同的模板编号（001, 002, ...）
   - 网页自动切换到对应的数据源

### 数据生成

使用 `processing_algorithms/gen_em_data_range1.py` 生成数据：

```bash
cd processing_algorithms
python gen_em_data_range1.py
```

输出目录：`data_templates/template_001/data/`

### 参数修改

如需修改仿真参数，编辑 `gen_em_data_range1.py`：

```python
# 修改网格大小
GRID_SIZE = 0.0008  # 改为0.0004可提高精度（数据量×4）

# 修改高度范围
HEIGHT_MAX = 400  # 改为600可增加层数

# 添加/删除信号源
EM_SOURCES = [
    # 在此列表中修改
]
```

## 扩展新数据源

### 创建新数据源模板

1. **复制模板**：
   ```bash
   copy data_templates/template_001 data_templates/template_002
   ```

2. **修改仿真脚本**：
   - 编辑 `processing_algorithms/gen_em_data_range1.py`
   - 修改 `EM_SOURCES`（信号源配置）
   - 修改 `SIM_BBOX`（仿真范围）
   - 修改 `POLYGON_COORDS`（边界多边形）

3. **生成数据**：
   ```bash
   cd processing_algorithms
   python gen_em_data_range1.py
   ```

4. **更新网页配置**：
   - 在 `electromagnetic.html` 中添加新的数据源选项
   - 示例：
   ```html
   <select id="dataSourceSelect" onchange="switchDataSource(this.value)">
     <option value="001">模板001: 标准城市环境</option>
     <option value="002">模板002: 新环境（自定义）</option>
   </select>
   ```

### 不同场景的参数建议

| 场景 | GRID_SIZE | GALT(m) | HEIGHT_MAX(m) | 特点 |
|-----|-----------|----------|--------------|-----|
| 城市密集区 | 0.0004 (44m) | 40 | 300 | 高精度，多遮挡 |
| 开阔郊区 | 0.0010 (110m) | 100 | 500 | 低精度，少遮挡 |
| 山区 | 0.0015 (165m) | 150 | 800 | 大范围，地形复杂 |
| 海岸线 | 0.0008 (88m) | 80 | 400 | 标准精度，混合环境 |

## 数据质量验证

### 检查脚本

使用 `processing_algorithms/check_em.py` 验证数据：

```bash
cd processing_algorithms
python check_em.py
```

### 验证项

1. **完整性**：所有层级文件是否存在
2. **格式正确**：JSON格式是否有效
3. **数据范围**：信号值是否在合理范围（-200 ~ -30 dBm）
4. **格点数量**：每层是否为预期数量
5. **边界一致性**：所有格点是否在边界多边形内

## 技术支持

### 文档参考

- **处理算法详解**: `processing_algorithms/README.md`
- **数据生成脚本**: `gen_em_data_range1.py`
- **检查工具**: `check_em.py`

### 常见问题

**Q: 数据生成太慢？**
A: 减小 `GRID_SIZE`（增大网格）或减小 `HEIGHT_MAX`（降低最大高度）

**Q: 信号强度不符合预期？**
A: 调整信号源功率（`power_w`）或路径损耗因子（`type_loss_factor`）

**Q: 想添加新的信号源类型？**
A: 在 `EM_SOURCES` 中添加新源，并更新 `SOURCE_TYPES` 配置
