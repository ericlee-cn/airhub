# MGS 低空仿真系统 - 整合版

## 系统概述
MGS（低空仿真系统）整合版，采用统一的Vue.js导航框架，将所有模块集中在一个入口页面，实现模块间的快速切换和统一管理。

## 核心功能

### 1. 统一导航系统
- **Vue.js驱动**：使用Vue 3作为前端框架，实现响应式导航
- **模块切换**：顶部导航栏支持在各模块间快速切换
- **分类筛选**：左侧侧边栏提供基础模型、数据资源、应用系统等分类
- **搜索功能**：支持关键词搜索模块名称、描述和分类
- **iframe加载**：各模块页面通过iframe加载，保持模块独立性

### 2. 模块分类

#### 基础模型
- **空域管理模型**：Cesium 3D空域可视化系统
- **地形环境模型**：DEM高程、卫星影像、三维地形
- **气象环境模型**：气象数据监测、天气趋势分析
- **电磁环境模型**：频谱分析、信号覆盖模拟

#### 数据资源
- **空域数据管理**：禁飞区、限高区、航路网络
- **地形数据管理**：DEM高程、卫星影像、地形模型
- **设备数据（CNS）**：通讯、导航、监视设备点位

#### 应用系统
- **分析工具**：数据统计、可视化分析、报告生成
- **飞行规则管理**：空域使用规则、审批流程
- **装备库管理**：飞行器、传感器、通信设备

### 3. 技术栈

#### 前端框架
- **Vue 3**: CDN版本 (https://cdn.jsdelivr.net/npm/vue@3/dist/vue.global.prod.js)
- **Iconify**: 图标库 (https://cdn.jsdelivr.net/npm/iconify-icon@3.1.0/dist/iconify-icon.min.js)
- **Axios**: HTTP客户端 (https://unpkg.com/axios/dist/axios.min.js)
- **TailwindCSS**: 内联样式

#### 第三方库
- **CesiumJS**: v1.114 - 3D地球可视化（空域、地形模块）
- **高德地图API**: 卫星影像服务
- **自定义CSS**: 玻璃态效果、渐变、动画

### 4. 界面设计

#### 配色方案
- **主色调**: #00aaff（科技蓝）
- **背景色**: #030a1c（深空蓝）
- **面板背景**: rgba(15, 23, 42, 0.9)（玻璃态）
- **边框色**: rgba(0, 170, 255, 0.2)

#### 视觉特性
- 玻璃态面板（backdrop-filter: blur）
- 渐变标题（bg-clip-text）
- 模块卡片悬浮效果
- 平滑过渡动画（transition）
- 响应式网格布局

## 快速开始

### 方式1：直接访问（推荐）
在浏览器中打开 `c:/mgs/basic/index.html`，无需服务器。

### 方式2：本地服务器（可选）
如需HTTP服务器，可运行：
```bash
cd c:/mgs/basic
python -m http.server 8000
# 或
npx http-server -p 8000
```
然后访问：http://localhost:8000

## 使用说明

### 导航操作
1. **顶部导航**：点击顶部导航栏的模块名称，切换到对应模块
2. **侧边栏分类**：点击左侧分类卡片，筛选显示对应类别的模块
3. **搜索**：在搜索框输入关键词，实时过滤模块列表
4. **查看详情**：点击模块卡片，右侧面板显示模块详细信息

### 模块操作
1. **直接打开**：点击"打开模块"按钮，在主内容区加载模块页面
2. **iframe切换**：在总览页面和模块页面之间切换，保持上下文

## 技术特点

### Vue.js优势
- **响应式数据绑定**：自动更新UI，无需手动DOM操作
- **组件化架构**：模块化设计，易于维护和扩展
- **计算属性**：自动计算过滤结果、状态颜色等
- **事件处理**：简洁的事件处理方法

### 模块独立性
- **iframe隔离**：各模块在独立iframe中运行，避免样式和脚本冲突
- **路由管理**：通过URL参数管理模块切换状态
- **状态共享**：可通过localStorage等共享数据（待实现）
