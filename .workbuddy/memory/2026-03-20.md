## 修复左侧目录折叠功能
修复了左侧资源树目录的折叠/展开功能初始状态问题：
- 一级目录（空域管理模型、三级航路网络层、固定航线、两级调度管理层、全要素保障层）：移除内容区的hidden类，箭头设置为0度（展开状态）
- 二级目录（通讯设备、导航设备、监视设备）：保持内容区的hidden类，箭头从Tailwind类`-rotate-90`改为内联样式`style="transform: rotate(-90deg);"`（折叠状态）
- JavaScript toggleSection函数逻辑正确，通过检测content.classList.contains('hidden')来切换显示/隐藏状态，并同步旋转箭头图标
- 问题根因：Tailwind的`-rotate-90`类会与JavaScript的内联样式设置冲突，改为纯内联样式控制旋转
